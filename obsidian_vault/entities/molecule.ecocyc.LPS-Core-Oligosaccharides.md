## Enriched Summary

Please note that this entity does not occur in this form in nature - it is assembled when already attached to Lipid-A "lipid A", and is found as part of the Lipopolysaccharides lipopolysaccharide structre. The core oligosaccharides are conceptually divided into two regions: inner core and outer core. The inner core is highly conserved, contains KDO (Kdo) and CPD-21358 (Hep) residues, and is often phosphorylated. The inner core oligosaccharide plays a critical role in essential barrier function of the outer membrane. The outer core comprises a tri-hexose backbone modified with varying side-branch substitutions of hexose and acetamidohexose residues. The outer core also provides an attachment site for the O-antigen.
